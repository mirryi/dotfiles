= Alacritty

include::../docs/variables.asciidoc[]

A package with Alacritty terminal configuration files.

This depends on the profile file {loaded}[`loaded.dhall`] in the parent
directory to set colors and font family. See
{profiletype}[`ProfileType.dhall`] for the profile configuration type.

== Package Dependencies

* link:../qt[`qt`]

== Dependencies

* https://github.com/alacritty/alacritty[`alacritty`]

== Local Configuration (Optional)

The file `local.dhall` can be created to set the font size, which defaults to `10`.
See {localtype}[`LocalType.dhall`] for more information

Example:

[source,dhall]
----
let LocalType = ./types/LocalType.dhall
in  LocalType::{ fontSize = 11 }
----
