[#local-config]
== Local Configuration

The file `{local}` can be created to set the music directories. See `{localtype}[]`.

Example:

[source,dhall]
----
let Export = ./types/LocalType.dhall

let export
    : Export
    = { library = "~/music/library" }

in  export
----
