[#local-config]
== Local Configuration

The file `{local}` must be created to set the `mpdnd` binary to call. See
`{localtype}[]`.

Example:

[source,dhall]
----
let Export = ./types/LocalExport.dhall

let export : Export = { bin = "mpdnd" }

in export
----
